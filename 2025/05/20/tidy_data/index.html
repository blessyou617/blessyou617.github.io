Tidy Data

A tidy data set is a standardized way of organizing data that makes it easier to manipulate, analyze, and visualize. The concept was popularized by Hadley Wickham, a prominent statistician and developer of the R programming language ecosystem.

Key Principles of Tidy Data

  1. Each variable forms a column.
    A variable is a measurable attribute (e.g., temperature, age, price).

  2. Each observation forms a row.
    An observation is a single instance or case (e.g., a person’s measurement at a given time).

  3. Each type of observational unit forms a table.
    Different entities (e.g., people vs. companies) should be in separate tables.

Example: Non-Tidy vs. Tidy

Non-tidy:

Name Math Science
Alice 85 90
Bob 78 88

Tidy:

Name Subject Score
Alice Math 85
Alice Science 90
Bob Math 78
Bob Science 88

Why Tidy Data Is Useful

  • Easier to manipulate using tools like pandas (Python) or tidyverse (R).
  • Compatible with most data visualization libraries.
  • Facilitates better data cleaning and transformation workflows.

🎻 Violin + Boxplot Faceted by Number of Cylinders (cyl)

This ggplot2 visualization shows how miles per gallon (mpg) varies by transmission type (am), with a separate plot (facet) for each cylinder count (cyl).

🧱 Code Breakdown

🧩 Explanation
🔹 ggplot(mtcars, aes(…))
Data: Using the built-in mtcars dataset.

Aesthetics:

x = factor(am): x-axis is transmission type (0 = automatic, 1 = manual), converted to a factor.

y = mpg: y-axis is miles per gallon.

fill = factor(am): fill color depends on transmission type.

🎻 geom_violin(…)
Purpose: Shows the full distribution of mpg for each am group.

trim = FALSE: do not cut the tails of the violin to match boxplot.

alpha = 0.6: 60% opacity (for transparency effect).

📦 geom_boxplot(…)
Purpose: Adds a boxplot inside each violin to show median and quartiles.

width = 0.1: makes the boxplot narrow, so it’s visible inside the violin.

fill = “white”: boxplot is white to stand out.

outlier.shape = NA: hides outlier dots.

🧱 facet_wrap(~ cyl)
Faceting: Creates one small plot per cylinder count (cyl).

Helps compare how mpg varies by am within each cylinder group.

🏷️ labs(…)
title: Sets the plot title.

x, y: Axis labels.

fill: Legend title for am (transmission type).
